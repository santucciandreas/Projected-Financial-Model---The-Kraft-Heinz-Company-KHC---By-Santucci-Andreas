# Kraft Heinz (KHC) – 5-Year Integrated Financial Model

## Overview

This project consists of a fully integrated three-statement financial model for The Kraft Heinz Company (NASDAQ: KHC), including:

- Income Statement
- Balance Sheet
- Cash Flow Statement

The model projects financial performance over a 5-year horizon using business-driven assumptions and provides a set of KPIs to evaluate growth, profitability, cash generation, solvency, shareholder returns, and investment risk.

## Project Structure

| Sheet | Description |
|---------|-------------|
| Cover & Index | Model metadata, navigation, and validation checks |
| Assumptions | Revenue drivers, margins, working capital, CAPEX, debt, and valuation assumptions |
| Financial Model | Integrated 3-statement forecast with balance-sheet reconciliation |
| Insights | KPI analysis and investment metrics |

## Tools Used

- Microsoft Excel 365
- Claude (Anthropic) for model structuring support
- Financial Modeling best-practice references (FMVA methodology)

## Key Assumptions

### Revenue Growth

Revenue is projected using a bottom-up approach by geographic segment:

| Segment | Growth Range |
|----------|-------------|
| North America | -1.0% to 1.5% |
| Developed International Markets | 1.0% to 2.0% |
| Emerging Markets | 3.0% to 4.0% |

### Margins

- Gross Margin: 34.7% → 35.7%
- SG&A as % of Sales: 13.99% → 13.5%
- Effective Tax Rate: 22%

### Capital Structure

- CAPEX: 4.0% of Sales
- Gradual debt reduction (~$200M annually)
- Conservative dividend growth policy

## Validation

The model includes automated balance checks for every projected period:

Assets = Liabilities + Equity

All financial statements are dynamically linked with no manual balancing plugs.

## Key Insights

- KHC appears to be a deleveraging and normalization story rather than a growth story.
- Free Cash Flow remains stable throughout the projection period.
- EBITDA margins recover significantly after the 2024 impairment impact.
- ROIC remains below the assumed WACC, suggesting continued economic value destruction under the base scenario.
