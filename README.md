# Kraft Heinz (KHC) – 5-Year Integrated Financial Model

## Overview

This project consists of a fully integrated three-statement financial model for The Kraft Heinz Company (NASDAQ: KHC), including:

- Income Statement
- Balance Sheet
- Cash Flow Statement

The model projects financial performance over a 5-year horizon using business-driven assumptions and provides a set of KPIs to evaluate growth, profitability, cash generation, solvency, shareholder returns, and investment risk.

An accompanying **interactive web dashboard** turns the model's KPIs into a visual summary that can be read in under four minutes (see below).

## Interactive Dashboard

A single-page, interactive dashboard built on top of the model's `Insights` sheet, designed for quick review by recruiters and analysts. It presents a headline read on the company, four key metrics, and five themed sections — each with an interactive chart and a short interpretive note.

**[▶ View the dashboard](./index.html)** — open `index.html`, or enable GitHub Pages.

| Section | Focus |
|---|---|
| 01 · Growth & Profitability | Net sales, EBITDA and margin recovery |
| 02 · Cash Generation | Free cash flow and conversion |
| 03 · Efficiency & Solvency | ROIC and net leverage (deleveraging) |
| 04 · Shareholder Returns | Dividends, EPS and payout ratio |
| 05 · Risk Analysis | Interest / dividend coverage |

A dashed gold line in every chart marks the boundary between **actuals (2022–2024)** and **projections (2025E–2029E)**. The dashboard is generated by `generate_dashboard.py`, which reads the `Insights` sheet and renders a self-contained `index.html` (charts via Plotly, loaded from CDN — no build step):

```bash
pip install openpyxl
python generate_dashboard.py "Projected_Financial_Model_-_The_Kraft_Heinz_Company__KHC__-_By_Santucci_Andreas.xlsx" index.html
```

## Key Assumptions

### Revenue Growth

Revenue is projected using a bottom-up approach by geographic segment:

| Segment | Growth Range |
|---|---|
| North America | -1.0% to 1.5% |
| Developed International Markets | 1.0% to 2.0% |
| Emerging Markets | 3.0% to 4.0% |

### Margins

- Gross Margin: 34.7% → 35.7%
- SG&A as % of Sales: 13.99% → 13.5%
- Effective Tax Rate: 22%

### Capital Structure

- CAPEX: 4.0% of Sales
- Gradual debt reduction (~$200M annually)
- Conservative dividend growth policy

## Validation

The model includes automated balance checks for every projected period:

- Assets = Liabilities + Equity
- All financial statements are dynamically linked with no manual balancing plugs.

## Key Insights

- KHC appears to be a **deleveraging and normalization story** rather than a growth story.
- Free Cash Flow remains stable throughout the projection period.
- EBITDA margins recover significantly after the 2024 impairment impact.
- ROIC remains below the assumed WACC, indicating that the company may continue to destroy economic value despite generating positive accounting profits under the base-case scenario.
